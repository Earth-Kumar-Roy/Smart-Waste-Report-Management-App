# Smart-Waste-Report-Management-App
This Smart Waste system is a full-spectrum digital tool that mobilizes civic action by allowing citizens to pinpoint sanitation problems with images and GPS data, streamlines the resolution workflow for field workers, and drives accountability through transparent, public statistics on cleanup performance.
